# [解集](https://leetcode-cn.com/problems/subsets/)

## 描述  
**中等**  

给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

说明：解集不能包含重复的子集。

**示例**

    输入: nums = [1,2,3]
    输出:
    [
        [3],
        [1],
        [2],
        [1,2,3],
        [1,3],
        [2,3],
        [1,2],
        []
    ]

## 解题  

借助上一题，[77](https://leetcode-cn.com/problems/combinations/)的回溯解法  

将这题看作，1个元素，2个元素，3个元素...的组合  


```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        res = []
        n = len(nums)
        
        def back(temp, k, i):
            if k==0:
                res.append(temp.copy())
                
            for j in range(i, n):
                temp.append(nums[j])
                back(temp, k-1, j+1)
                temp.pop()
        
        for k in range(n+1):
            back([], k, 0)
        
        return res
```

向后遍历，碰到一个数就使现有的子集加上这个数，在添加到解集中

```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        res = [[]]

        for i in range(len(nums)):
            for sub in res[:]:             
                res.append(sub+[nums[i]])
    
        return res

```
其中，res[:]是一个必要的trick  

因为在循环中res是在不断变化的，使用res[:]可以获得一段res的不变化的内容
